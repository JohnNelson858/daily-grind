About This Page

This page dynamically generates a different coffee for each day of the week using JavaScript.

Data is stored inside JavaScript objects that contain a unique image, alt tag, description, color, and coffee name for each weekday.

The page changes dynamically using JavaScript along with document.getElementById() and querySelector().

Clicking the weekday links passes data through the query string using URLSearchParams(), allowing all seven coffee specials to appear using only one webpage.

At LargeBuck$ Coffee we believe every day deserves its own special flavor and personality. Our rotating daily coffee menu gives customers something fun and unique to look forward to throughout the week.

Whether you enjoy rich espresso drinks, refreshing cold brews, or sweet seasonal flavors, our coffee shop has something for everyone.
